# FoodMenu marketing site

This branch (marketing-landing-page) contains a simple static marketing landing page for the FoodMenu project.

What I added:

- index.html — marketing landing page (hero, features, how-it-works, pricing, footer)
- styles.css — lightweight responsive styling
- assets/logo.svg — simple SVG logo placeholder
- .github/workflows/pages.yml — GitHub Actions workflow to deploy the branch to GitHub Pages

Deploying to GitHub Pages

The included workflow will upload the branch contents as a Pages artifact whenever you push to marketing-landing-page. GitHub will then deploy it to Pages. It can take a minute or two for the site to appear.

After the workflow runs successfully, your site will be available at:

https://Vuthi1111.github.io/Foodmenu-Marketing/

If you prefer to deploy manually, you can also:
- Go to the repository Settings → Pages and select the branch `marketing-landing-page` as the source.

Notes

- This is a showcase site only — no signup forms or analytics are included.
- Update index.html and styles.css on the branch to change copy, colors, or images. If you want, I can open a pull request from this branch into main/master when you're ready.
